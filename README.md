# Personal Navigation Site

Junjie's personal navigation website, a concise and efficient collection of personal links.

## Project Overview

This is a personal navigation website built with static HTML and developed using Gulp build tool. The website features a clean design style and provides quick access to personal-related links.

## Tech Stack

- HTML5
- CSS3  
- JavaScript (ES6+)
- Gulp (Build Tool)
- Browser-sync (Development Server)

## Development Guide

### Requirements

- Node.js
- pnpm

### Install Dependencies

```bash
pnpm install
```

### Development Mode

Start development server with hot reload:

```bash
pnpm run dev
```

### Build Production

Generate optimized production files:

```bash
pnpm run build
```

## Project Structure

```
src/
```
